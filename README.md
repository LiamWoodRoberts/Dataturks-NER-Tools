# Dataturks-NER-Tools

A repo with tools for working with Dataturks NER annotation service. Services include:
- Formatting unannotated text for upload to Dataturks.
- Automated training of CRF models using only an annotated Dataturks file.
- Automated metric tracking for every model trained.
- Generating Predictions using CRF models.
- Formatting model predictions to allow "Pre-Annotation" upload to Dataturks and speed up labelling process.
